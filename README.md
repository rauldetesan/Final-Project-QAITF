# Final-Project-QAITF
# Se instaleaza Google Chrome
# Se instaleaza Pycharm Community Edition : https://www.jetbrains.com/pycharm/download
# Se instaleaza Git : https://git-scm.com/downloads
# Se deschide gitBash cu click dreapta in locatia unde se doreste a fii clonat
# Se scrie comanda git clone https://github.com/rauldetesan/Final-Project-QAITF.git	
# In Pycharm deschidem proiectul: File > Open > Folderul proiectului
# In terminal instalam librariile necesare: - pip install behave
#                                           - pip install selenium
#                                           - pip install webdrivermanager
#                                           - pip install webdriver-manager
#                                           - pip install behave-html-formatter
# Testarea cu generarea raportului se ruleaza din terminal folosind comanda: behave -f html -o behave-report.html
# Accesarea raportului se face cu, click dreapta pe raport, apoi Open In > Browser > Chrome
